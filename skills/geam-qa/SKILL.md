# GEAM 影像系统问答技能

## 能力矩阵

| 能力 | 触发关键词 | 读取 |
|------|-----------|------|
| 权限申请 | "申请权限" "开通" "添加权限" | permission.md |
| 登录问题 | "登录不了" "进不去" "没权限" | login.md |
| 通道管理 | "申请通道" "新增通道" "删除通道" | channel.md |

## 对话流程

### 分支逻辑

```
用户问题
  ├── 含"权限"关键词 → 读取 permission.md
  │   ├── 财务岗 → 提供申请表
  │   └── 非财务岗 → 告知只能财务申请
  ├── 含"登录"或"没权限" → 读取 login.md
  │   ├── 有历史记录 → 复用历史方案
  │   └── 无历史 → 知识库查询
  ├── 含"通道" → 读取 channel.md
  │   └── 按流程处理
  └── 其他 → 询问具体需求
```

### 转人工条件

- 用户问题超过3轮未解决
- 涉及账号异常等需后台排查
- 用户明确要求人工处理

## Error Handling

| 错误情况 | 处理方式 |
|---------|---------|
| 非财务岗申请财务权限 | "抱歉，该权限只能财务岗位申请" |
| 需求不明确 | 继续追问，直到能匹配到具体权限类型 |
| 截图无法识别 | 请用户重新提供清晰的截图或文字描述 |
