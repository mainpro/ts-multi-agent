## 签字要求

| 权限类型 | 签字要求 | 岗位限制 |
|---------|---------|---------|
| 凭证查询权限 | 直线经理签字 | 仅财务岗 |
| 资产类UG/UB | 财务签字 + 直线领导签字 | 无限制 |
| 财务签收权限 | 直线经理签字 | 仅财务岗 |
| 财司回单权限 | 直线经理签字 + 财务部长会签 | 仅财务岗 |
| 凭证后补权限 | 直线经理签字 | 仅财务岗 |
| 新增法人公司 | 财务签字 + 直线领导签字 | 无限制 |

## 回复模板

### 财务岗位限制权限回复
```
您好，该权限只能是财务来申请，首先确认您是否是财务岗？
如果不是财务岗无法帮您添加。
如果您是财务，请下载下面的申请表，填写后由您的直线经理签字，之后把填写的申请表和签字文件发给我，我这边给您找到对应工程师来添加。

申请表下载：[对应链接]
```

### 无岗位限制权限回复
```
您好，开通相关权限需要填写申请表，并由财务和直线领导签字后提交。

申请表下载：[对应链接]

请将填写好的申请表和签字文件发给我，我会转交给工程师处理。
```

### 回单权限特殊回复（需要双会签）
```
您好，回单查询的权限只能财务岗位来申请，请填写权限申请表，直线经理和财务部长都需要做一下会签。

申请表下载：[对应链接]

请将填写好的申请表和会签文件发给我，我会转交给工程师处理。
```
