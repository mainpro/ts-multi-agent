# GEAM 权限申请处理详情

## 处理流程

### Step 1: 识别用户意图

当用户说"我要申请权限"或类似表述时：

**反问**："您好，请问您要登录系统操作什么呢？"

### Step 2: 根据回答匹配权限类型

| 关键词 | 权限类型 | 岗位限制 |
|-------|---------|---------|
| 凭证查询/下载 | 凭证查询 | 仅财务岗 |
| 资产、UG、UB | 资产类扫描 | 无限制 |
| 签收 | 签收权限 | 仅财务岗 |
| 回单 | 回单查询 | 仅财务岗 |
| 凭证后补/笔记 | 后补权限 | 仅财务岗 |
| 增加公司/新增公司 | 公司权限 | 无限制 |

### Step 3: 岗位确认与申请表提供

#### 财务岗位限制权限

**确认用户是否为财务岗：**
```
您好，该权限只能是财务来申请，首先确认您是否是财务岗？
如果不是财务岗无法帮您添加。
如果您是财务，请下载下面的申请表，填写后由您的直线经理签字，之后把填写的申请表和签字文件发给我，我这边给您找到对应工程师来添加。
```

#### 无岗位限制权限

```
您好，开通相关权限需要填写申请表，并由财务和直线领导签字后提交。

申请表下载：[对应链接]

请将填写好的申请表和签字文件发给我，我会转交给工程师处理。
```

### Step 4: 特殊情况处理

#### 用户说"复制同事权限"

**回复**：
```
抱歉系统目前不提供权限复制的功能，您可以问一下您同事之前在系统里都操作哪些工作，告诉我我来告诉您如何申请。
```

#### 用户提供签字文件

→ 直接转人工，由工程师处理

## 签字要求汇总

| 权限类型 | 签字要求 |
|---------|---------|
| 凭证查询权限 | 直线经理签字 |
| 资产类UG/UB | 财务签字 + 直线领导签字 |
| 财务签收权限 | 直线经理签字 |
| 财司回单权限 | 直线经理签字 + 财务部长会签 |
| 凭证后补权限 | 直线经理签字 |
| 新增法人公司 | 财务签字 + 直线领导签字 |

## 常见场景示例

### 场景1：用户直接说"我要申请权限"

1. 反问用户具体需求
2. 根据回答匹配权限类型
3. 提供对应申请表

### 场景2：用户说"笔记下载/填写/提交"

1. 确认用户是否为财务岗位
2. 提供凭证后补申请表
3. 告知需要填写并由直线经理签字

### 场景3：用户提供签字文件

→ 直接转人工，由工程师处理
