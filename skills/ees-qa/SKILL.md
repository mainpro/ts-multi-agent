# EES 报销系统问答技能

## 能力矩阵

| 能力 | 触发关键词 | 读取 |
|------|-----------|------|
| 发票格式问题 | "格式错误" "上传失败" "不支持" | invoice-format.md |
| 报销提交问题 | "提交不了" "提交失败" "报销异常" | submit-issue.md |
| 报销进度 | "进度" "到哪了" "什么时候到账" | approval-status.md |
| 报销流程 | "怎么报销" "流程" "需要什么材料" | reimburse-process.md |

## 对话流程

### 分支逻辑

```
用户问题
  ├── 截图含"发票格式错误" → 读取 invoice-format.md
  │   ├── 首次问题 → 提供解决方案
  │   └── 有历史记录 → 复用历史方案
  ├── 报销提交失败 → 读取 submit-issue.md
  │   ├── 格式问题 → 引导改格式
  │   ├── 审批异常 → 转人工
  │   └── 其他 → 询问详情
  ├── 询问进度 → 读取 approval-status.md
  │   └── 指导查询方法
  ├── 询问流程 → 读取 reimburse-process.md
  │   └── 提供流程说明
  └── 其他 → 询问具体需求
```

### 转人工条件

- 用户更换格式后仍失败
- 涉及审批流程异常
- 用户明确要求人工
- 超过3轮未解决

## 快速参考

| 问题类型 | 常见原因 | 解决方式 |
|---------|---------|---------|
| 发票格式错误 | JPG/PNG不支持，只支持PDF/XML/OFD | 引导更换格式 |
| 提交失败 | 审批流程异常、附件问题 | 转人工排查 |
| 进度缓慢 | 等待审批 | 告知预计时间 |

## Error Handling

| 错误情况 | 处理方式 |
|---------|---------|
| 需求不明确 | 继续追问，让用户补充信息或截图 |
| 无法识别截图 | 请用户重新提供清晰的截图或文字描述 |
| 问题无法解决 | 转人工处理 |
