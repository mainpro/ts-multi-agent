# 审批进度查询

## 查询方式

1. **在线查询**：登录EES系统，在"报销记录"中查看
2. **手机端查询**：通过Ihaier或财务云移动端查看

## 审批状态说明

| 状态 | 说明 |
|-----|------|
| 待审批 | 已提交，等待审批人处理 |
| 审批中 | 正在审批流程中 |
| 审批通过 | 已通过审批，等待财务打款 |
| 审批拒绝 | 被审批人拒绝，需要修改后重新提交 |
| 已打款 | 报销款项已打款到员工账户 |

## 审批流程

1. 部门负责人审批
2. 财务部门审批
3. 总经理审批（如需要）
4. 财务打款

## 常见问题

- **审批时间**：一般3-5个工作日
- **打款时间**：审批通过后1-3个工作日
- **进度缓慢**：可联系财务部门咨询
