## 执行方式

### API测试

```bash
curl -X POST http://localhost:3000/tasks/stream \
  -H "Content-Type: application/json" \
  -d '{"userId": "test-xxx", "sessionId": "test-xxx", "requirement": "测试内容"}'
```

### 验证要点

1. **意图分类正确性** - 返回的intent是否符合预期
2. **技能匹配准确性** - matchedSkills是否正确
3. **多任务拆分** - 多个问题是否正确拆分为多个任务
4. **转人工触发** - 符合条件时是否正确转人工
5. **会话继承** - 同一sessionId是否继承上下文
